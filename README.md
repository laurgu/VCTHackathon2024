# VCTHackathon2024

## Setting up the ENV file

The various files mix in the Gemini API to simulate an LLM response and AWS SDK interactions. Below is how environment variables are set up in the .env file:
```
GOOGLE_API_KEY=

AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_DEFAULT_REGION=

BEDROCK_AGENT_ID=
AGENT_ALIAS_ID=
```
